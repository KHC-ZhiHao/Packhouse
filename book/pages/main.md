# Packhouse

## 安裝

npm
```bash
npm i packhouse
```

web
```html
<script src="https://khc-zhihao.github.io/Packhouse/dist/index.js"></script>
<script>
    let factory = Packhouse.createFactory()
</script>
```

webpack
```js
import Packhouse from 'packhouse'
let factory = Packhouse.createFactory()
```

nodejs
```js
let Packhouse = require('packhouse')
let factory = Packhouse.createFactory()
```
